# PSC+TS Forms Application

Aplicação de formulários para PSC+TS e PSC Consórcios com tema preto e dourado.

## 🎯 Características

- **25 formulários** no total
  - 11 PJ PSC+TS
  - 12 PF PSC+TS
  - 2 PSC Consórcios (com logo específica)
- Tema preto e dourado premium
- Links diretos para cada formulário
- Formatação profissional para WhatsApp
- Logo dinâmica (PSC+TS ou PSC Consórcios)

## 📁 Estrutura

- `index.html` - Página principal
- `styles.css` - Estilos (tema preto e dourado)
- `script.js` - Lógica e configuração de formulários
- `logo.png` - Logo PSC+TS
- `logo-psc.png` - Logo PSC Consórcios
- `form-links.txt` - Lista de links diretos

## 🚀 Como Usar

1. Abra `index.html` em um navegador
2. Selecione a categoria (PJ ou PF)
3. Escolha o serviço desejado
4. Preencha o formulário
5. Envie via WhatsApp

## 🔗 Links Diretos

Cada formulário pode ser acessado diretamente via URL:
```
index.html?form=<form-id>
```

Veja `form-links.txt` para lista completa.

## 📝 Observações

- Formulários PSC Consórcios usam logo específica
- Todos os endereços incluem CEP
- WhatsApp formatado de forma profissional
- Observações específicas por tipo de serviço
